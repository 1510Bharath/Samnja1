# Samjna Web — V2 (Local / No Encryption)

## Run locally
```
pip install -r requirements.txt
python server.py
# Open Chrome at http://localhost:8000
```

## Deploy to cloud
```
./deploy.sh        # on Ubuntu server
```
Place Lady1.jfif in static/img/ before running.
Recordings saved to: D:\Samjna\Claude\MultiModal_Stress\Output  (Windows)
                 or: /app/uploads  (Docker/Linux)
